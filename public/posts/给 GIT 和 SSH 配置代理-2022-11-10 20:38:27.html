给 GIT 和 SSH 配置代理
发布时间：2022-11-10
归档分类：
  • 教程记录
标签：
  • git
  • ssh

今天 push 不了项目，估计又是网络的问题，决定还是给git和ssh配置一下代理，不然太难受了。 配置记录如下： 1.使用socks5代理，我这边的端口是1089

git config --global http.https://github.com.proxy socks5://127.0.0.1:1089

2.修改 ~/.ssh/config 文件

Host github.com
    User git
    ProxyCommand nc -v -x 127.0.0.1:1089 %h %p
